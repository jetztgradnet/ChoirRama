# Vowel Mappings

The vowel is selected by sending a `CC3` MIDI message. The following table shows known vowels for CC values:

_**note: this table is tbd**_

|CC value |Vowel|
|---------|-----|
| 0-22    | Ah  |
| 23-27   | Eh  |
| 28-52   | Oh  |
| 53-67   | Ah  |
| 68-90   | Eee |
| 91-105  | Eh  |
| 106-112 | Uhh |
| 113-120 | Umm |
| 121-126 | eh  |
